- Dark zinc theme

## Improvement Areas

### 1. Animations (Framer Motion)
- **Page transitions** — Smooth page load
- **Card animations** — Staggered entrance
- **Micro-interactions** — Button hovers, clicks
- **Drag animations** — Smooth drag feedback
- **Status changes** — Animate when moving between columns

### 2. Visual Polish
- **Glassmorphism** — Subtle blur effects on cards
- **Gradients** — Subtle background gradients
- **Shadows** — Layered shadows for depth
- **Borders** — Subtle gradients on borders
- **Icons** — More cohesive icon set

### 3. Loading States
- **Skeleton loaders** — Instead of "Loading..."
- **Shimmer effect** — While data fetches
- **Progressive loading** — Show content progressively

### 4. Feedback
- **Toast notifications** — Success/error toasts
- **Optimistic updates** — Instant UI feedback
- **Hover states** — Rich hover interactions
